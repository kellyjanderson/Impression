# Loft Spec 50 Test: Simple Correspondence Regression Fixtures

## Overview

This test specification defines verification for simple loft correspondence regression fixtures.

## Backlink

- [Loft Spec 50: Simple Correspondence Regression Fixtures (v1.0)](../specifications/loft-50-simple-correspondence-regression-fixtures-v1_0.md)

## Automated Smoke Tests

- simple rectangular and cylindrical fixtures loft successfully through the surfaced path
- reference images and STL artifacts can be generated for both fixtures

## Automated Acceptance Tests

- rectangular fixtures include a geometric check that reveals twist drift
- cylindrical fixtures include a geometric check that reveals deformation drift
- representative reference images and STL artifacts detect visible output change
