# Surface Spec 56 Test: Surface Canonical Promotion Criteria

## Overview

This test specification defines verification for canonical promotion of
surfaces.

## Backlink

- [Surface Spec 56: Surface Canonical Promotion Criteria (v1.0)](../specifications/surface-56-surface-canonical-promotion-criteria-v1_0.md)

## Manual Smoke Check

- Review the canonical-promotion prerequisites against current surfaced
  coverage.
- Confirm the remaining mesh role is compatibility-only rather than hidden
  canonical execution.

## Automated Smoke Tests

- promotion prerequisites are explicit
- tolerated compatibility debt is explicit

## Automated Acceptance Tests

- non-negotiable promotion conditions are explicit
- promotion criteria align with migration ordering and verification burden
- no hidden mesh-first execution remains on canonical surfaced paths
