# Surface Spec 125 Test: Standalone Mesh Utility Tool Contract

## Overview

This test specification defines verification for retained standalone mesh
utilities.

## Backlink

- [Surface Spec 125: Standalone Mesh Utility Tool Contract (v1.0)](../specifications/surface-125-standalone-mesh-utility-tool-contract-v1_0.md)

## Automated Smoke Tests

- retained standalone mesh utilities are identified explicitly
- standalone mesh utilities are described as tools rather than canonical modeling APIs

## Automated Acceptance Tests

- mesh CSG, hull, or similar utilities are only retained when they have explicit analysis/repair/debugging purpose
- utility/tool posture remains separate from the surfaced modeling contract
