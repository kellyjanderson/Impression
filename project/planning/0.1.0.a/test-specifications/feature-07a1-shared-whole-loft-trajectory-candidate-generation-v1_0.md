# Feature Spec 07A1 Test: Shared Whole-Loft Trajectory Candidate Generation

## Overview

This test specification defines verification for whole-loft shared trajectory
candidate generation.

## Backlink

- [Feature Spec 07A1: Shared Whole-Loft Trajectory Candidate Generation (v1.0)](../specifications/feature-07a1-shared-whole-loft-trajectory-candidate-generation-v1_0.md)

## Automated Smoke Tests

- representative loft evidence produces shared whole-loft trajectory candidates
- candidate outputs remain inspectable and durable

## Automated Acceptance Tests

- generation remains limited to whole-loft shared trajectories
- candidate output shape remains stable for identical inputs
- fitted curve evidence contribution remains explicit
