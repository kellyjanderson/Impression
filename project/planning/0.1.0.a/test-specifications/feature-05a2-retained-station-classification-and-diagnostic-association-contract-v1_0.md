# Feature Spec 05A2 Test: Retained Station Classification and Diagnostic Association Contract

## Overview

This test specification defines verification for retained station-class
recording and diagnostic association.

## Automated Smoke Tests

- reduced progression results retain topology and hidden control station
  classifications
- supporting diagnostic references remain inspectable from retained station
  records

## Automated Acceptance Tests

- retained station-class recording is explicit
- diagnostic association does not blur topology and control station classes
- retained structure remains inspectable after reduction
