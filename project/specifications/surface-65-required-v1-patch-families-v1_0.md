# Surface Spec 65: Required V1 Patch Families (v1.0)

## Overview

This specification defines the patch families that are mandatory for the first
surface-first implementation program.

## Backlink

Parent specification:

- [Surface Spec 22: V1 Patch Family Scope and Explicit Exclusions (v1.0)](surface-22-v1-patch-family-scope-v1_0.md)

## Scope

This specification covers:

- the required v1 family list
- minimum support obligations for each required family
- justification boundary for inclusion

## Behavior

This branch must define:

- which patch families are required in v1
- what kernel obligations each required family must satisfy
- why each included family is necessary for the planned feature set

## Constraints

- the included set must be explicit and bounded
- each included family must map to concrete feature coverage
- the required set must remain achievable without a full CAD kernel

## Acceptance

This specification is complete when:

- the required v1 family list is explicit
- support obligations per family are explicit
- inclusion rationale is explicit
