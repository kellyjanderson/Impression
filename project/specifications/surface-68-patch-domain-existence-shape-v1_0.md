# Surface Spec 68: Patch Domain Existence and Shape Contract (v1.0)

## Overview

This specification defines whether every patch has a parameter domain and what
general shape that domain takes.

## Backlink

Parent specification:

- [Surface Spec 23: Patch Parameter-Domain Contract (v1.0)](surface-23-patch-parameter-domain-contract-v1_0.md)

## Scope

This specification covers:

- universal domain existence
- canonical domain shape
- family-specific variation boundaries

## Behavior

This branch must define:

- whether every patch is required to expose a parameter domain
- the abstract shape of that domain
- what limited family-specific variation is permitted

## Constraints

- domain existence rules must be explicit
- domain shape must be broad enough for all required v1 families
- family-specific variation must remain bounded
