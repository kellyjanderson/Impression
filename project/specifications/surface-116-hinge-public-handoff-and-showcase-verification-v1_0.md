# Surface Spec 116: Hinge Public Handoff and Showcase Verification (v1.0)

## Overview

This specification defines the public handoff, reference evidence, and showcase burden for surfaced hinges.

## Backlink

- [Surface Spec 104: Surface-Native Hinge Replacement (v1.0)](surface-104-surface-native-hinge-replacement-v1_0.md)

## Scope

This specification covers:

- public hinge API migration posture
- surfaced hinge preview/export handoff
- showcase and reference artifact requirements

## Behavior

This branch must define:

- how surfaced hinge outputs are handed to consumers
- what deprecation or compatibility posture remains for legacy mesh hinge APIs
- what showcase/reference fixtures must exist before hinge promotion

## Constraints

- surfaced hinge promotion must not happen without examples, docs, and reference artifacts
- public migration posture must remain explicit while legacy APIs still exist
- hinge outputs must remain consumer-compatible through standard boundary tessellation only
