# Surface Spec 101: Surface-Native Text Replacement (v1.0)

## Overview

This specification defines the surface-first replacement path for deprecated
mesh-primary text generation.

## Backlink

- [Surface Spec 99: Surface-Native Replacement Program (v1.0)](surface-99-surface-native-replacement-program-v1_0.md)

## Scope

This specification covers:

- text outline to topology-native staging
- surface-native raised, inset, or embossed text emission
- public replacement posture for deprecated mesh text APIs

## Behavior

This branch must define:

- the canonical surface-first output of text modeling
- how text profiles become surface-native bodies or consumers
- how text color and placement flow without making mesh the primary output

## Constraints

- text replacement must preserve topology-native outline stages where useful
- text replacement must terminate in surface-native outputs rather than mesh
- public text replacement must have durable usage docs

Final.

## Acceptance

This specification is complete when:

- a surface-first text replacement contract exists for the deprecated text capability
- verification requirements are defined by its paired test specification
- documentation requirements for the public text replacement are explicit
