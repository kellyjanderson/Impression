# Visual Rules

## Ask These Questions

* What gets attention first?
* What creates grouping?
* What establishes contrast?
* Where is the visual weight concentrated?
* What is decorative versus functional?

## Strong Defaults

* Prefer one primary emphasis layer, not many.
* Use consistent alignment lines.
* Use whitespace to separate groups before adding borders.
* Avoid equal emphasis on all elements.
* Keep decorative treatment subordinate to task clarity.
