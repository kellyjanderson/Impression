# Accessibility Basics

## Minimum Checks

* keyboard-only navigation works
* focus is visible
* text contrast is sufficient
* important non-text indicators are perceivable
* color is not the sole cue
* touch and pointer interaction are not excessively fragile

## Good Defaults

* keep focus rings obvious
* make state visible with multiple cues
* avoid tiny hit targets
* do not hide essential controls behind hover-only access
