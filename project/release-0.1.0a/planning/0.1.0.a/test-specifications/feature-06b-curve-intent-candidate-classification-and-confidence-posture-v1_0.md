# Feature Spec 06B Test: Curve-Intent Candidate Classification and Confidence Posture

## Overview

This test specification defines verification for curve-intent candidate
classification and confidence posture.

## Backlink

- [Feature Spec 06B: Curve-Intent Candidate Classification and Confidence Posture (v1.0)](../specifications/feature-06b-curve-intent-candidate-classification-and-confidence-posture-v1_0.md)

## Automated Smoke Tests

- representative evidence produces candidate intent reports or explicit
  indeterminate posture
- confidence or uncertainty remains inspectable

## Automated Acceptance Tests

- weak evidence is not overstated as strong inferred intent
- indeterminate or refusal posture remains a first-class output
- candidate reporting stays aligned with descriptor evidence
