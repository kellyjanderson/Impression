# Learned From Doing

This folder captures durable research notes about what was learned by actually
executing work, especially when the work exposed process failures, hidden
assumptions, or practical constraints that were not obvious from planning
alone.

These notes are not directives by themselves.

They exist so we can:

- preserve execution-time lessons
- separate observed reality from proposed process rules
- use those lessons later when composing directives, specs, or workflow policy
