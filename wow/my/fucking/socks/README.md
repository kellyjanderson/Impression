# WOW Demo

One-command preview mode:

```bash
impression preview wow/my/fucking/socks/off.py
```

This demo file currently disables the measurement cage in preview and keeps only the main foreground scene for smoother playback.

Standalone real-time PyVista mode (no Impression preview window):

```bash
python wow/my/fucking/socks/off_standalone.py
```

Optional controls:

```bash
python wow/my/fucking/socks/off_standalone.py --duration 180 --interval 1.0
```
