# Potential Directives

This folder is reserved for agent directives that are still being developed and
tested.

Treat the contents like any other agent directive while working in this
directory, but keep the following restrictions explicit:

- never add these directives to any skill set
- never add these directives to any published directive set
- never treat these directives as canonical
- only promote them by explicitly moving them out of this folder into a folder
  without this restriction

For the workspace containing this `.agents/` folder, agents should follow these
directives as local workspace authority when they are relevant, while still
treating them as non-canonical and non-publishable until they are explicitly
moved out of `potentials/`.
