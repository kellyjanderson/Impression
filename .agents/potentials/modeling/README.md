# Potential Modeling Directives

This folder holds non-canonical modeling directives that are being developed
and tested in this workspace.

They should be treated as local workspace directives only. They must not be
added to a skill set, a published directive set, or otherwise treated as
canonical unless they are explicitly moved out of `potentials/`.

## Documents

- [How To Model](how-to-model.md)
  Modeling workflow guidance for using visual feedback, sub-agent critique,
  reference images, and local research during model creation.
- [Model Planning And Specs](model-planning-and-specs.md)
  How model-plan documents should behave, how modeling specs decompose work,
  and how multi-part modeling scope should be surfaced early instead of being
  hidden inside one oversized example task.
- [Detail And Completeness](detail-and-completeness.md)
  A potential framework for expressing modeling detail as a scale ratio and for
  deciding when model-spec work is complete enough to stop refining.
