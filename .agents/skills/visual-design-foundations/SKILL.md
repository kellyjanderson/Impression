# Visual Design Foundations

Use this skill when an agent needs durable visual design judgment rather than ad hoc styling.

## Use This For

* composition and visual hierarchy
* spacing, alignment, and grouping
* emphasis and restraint
* typography scale and scanning flow
* evaluating whether a design feels intentional and legible

## Core Principles

* use scale to signal importance
* use hierarchy to control reading order
* use balance to distribute visual weight intentionally
* use contrast to separate levels and states
* use grouping to make relationships obvious
* use whitespace as structure, not leftover space

## Workflow

1. Identify what should receive attention first.
2. Make grouping and hierarchy legible through placement, spacing, and scale.
3. Remove unnecessary emphasis so the primary signal stays clear.
4. Check alignment, rhythm, and negative space.
5. If color carries structural meaning, pair with a color-system or accessibility pass.

## Output Shape

Prefer a short visual audit with:

* primary focal point
* hierarchy plan
* grouping and spacing notes
* contrast and emphasis notes
* clutter or restraint risks
