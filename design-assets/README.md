# Design Assets

This folder holds scene models used to render documentation images.
Models here are rendered by Impression during the docs build and the output
images are stored under `docs/images/renders/`.

Structure:
- `design-assets/models/<feature>/<scene>.py`
- `docs/images/renders/<feature>/<scene>.png`

Note: design-assets models are not necessarily runnable examples and may use
higher resolution settings for cleaner renders.
