# AIgents Folder Contract

This folder is the repository-local instruction/template source for AI agents.

## Agent Rule

Before taking action on a task, review `AIgents/` for relevant instructions and templates.

## Current Contents

- `release_template.md`: required structure/policy for release notes.
- `development_guidance.md`: development-facing rules for agents working on Impression itself.

## Maintenance

- Keep only active instruction/template files here.
- Remove outdated guidance instead of keeping conflicting copies.
